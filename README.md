# Quiz Master

Quiz Master is an interactive quiz game built with **JavaScript (ES6+)** using **Object-Oriented Programming (OOP)** principles. The application fetches quiz questions from a REST API and provides a complete quiz experience with dynamic questions, a countdown timer, sound effects, keyboard shortcuts, and a persistent leaderboard.

## Features

* Dynamic quiz generation using a REST API.
* Select the quiz category, difficulty, and number of questions.
* Loading and error handling while fetching data.
* Support for Multiple Choice and True/False questions.
* Answer questions using either the mouse or keyboard shortcuts (1–4).
* 15-second countdown timer for every question.
* Randomized answer order.
* Real-time score tracking.
* Accuracy percentage calculation.
* High Score system with a Top 10 leaderboard.
* Persistent leaderboard using Local Storage.
* Sound effects for correct answers, wrong answers, countdown, timeout, and game completion.
* Responsive and user-friendly interface.

2. Open the project folder.


3. Run the project using **Live Server** or any local development server.
